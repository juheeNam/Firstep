# Firstep - 사이드프로젝트 런치패드

> "두루뭉실한 아이디어만 있고 어디서 시작해야 할지 몰랐던 나를 위해 만들었습니다. Firstep은 AI가 아이디어를 단계별 개발 로드맵으로 만들어주고, 완료까지 옆에서 가이드합니다."

## 서비스 소개

Firstep은 사이드프로젝트를 처음 시작하는 1인 개발자가 아이디어를 실제 서비스로 완성할 수 있도록 돕는 AI 기반 프로젝트 관리 웹앱이다.

## 타겟

- 사이드프로젝트를 처음 혼자 시작하는 개발자
- 아이디어는 있지만 기획/개발 순서를 모르는 사람
- 1인 개발 시 기술 스택 선택이 막막한 사람

## 해결하는 문제

1. "뭐부터 해야 하지?" → 기획, 설계, 개발, 배포 순서를 몰라 시작을 못 함
2. "혼자 하다 흐지부지" → 동기부여가 떨어지고 진행이 멈춤
3. "서버, DB, 배포 뭐 써야 해?" → 1인 개발에 맞는 기술 스택 정보 부족

## 회원 관리

- 구글 로그인만 지원 (개인정보 처리 부담 최소화)
- 추후 GitHub 로그인 추가 검토

## 1단계 MVP

### 온보딩 플로우

구글 로그인 → 스택 온보딩(선택, 스킵 가능) → 아이디어 있어요/없어요 선택 → AI 대화로 기획 구체화 → 로드맵 생성

온보딩 이탈 방지:
- 스택 선택은 선택사항
- 스킵해도 서비스 이용 가능
- 로드맵 생성 후 넛지: "스택을 추가하면 더 정확한 추천을 받을 수 있어요"

### AI 기획 구체화 + 로드맵 생성

두 가지 진입 케이스:
- 아이디어 있는 경우 → 5지선다 질문으로 구체화 → 로드맵 생성
- 아이디어 없는 경우 → 페인포인트 발굴 질문부터 시작 → 대화를 통해 아이디어 도출 후 로드맵 생성

AI 대화 한도:
- 토큰 기반 제한
- UI는 게이지 바 + 숫자 병행 표시
- 무료/PRO 플랜 간 차등 제공 (수치는 베타 후 결정)
- 기획 대화 단계에서만 AI 대화 가능
- 로드맵 확정 후에는 막힌 스텝 힌트 용도로만 사용

### 로드맵 구조

블록(큰 단위) > 투두(작은 단위)

블록 커스터마이징:
- 순서 드래그 앤 드롭
- 블록 추가/삭제/이름 수정 (토큰 소모 없음)
- AI로 블록 재생성 → 토큰 N/5 소모

투두 기준:
- 로컬에서 실행해서 눈으로 확인할 수 있는 단위
- 30분~2시간 안에 끝낼 수 있는 크기
- 순서 강제 없음
- 투두 추가/삭제 가능

### 빌드 로그 감성 대시보드

- DONE/RUN/TODO 상태 표시
- 전체 진행률 시각화
- 연속 개발일 스트릭
- 오늘 완료한 투두 수
- AI 토큰 잔여량 게이지

### 이메일 넛지

- 3일 연속 미접속 시 자동 이메일 발송
- 수신 거부 옵션 제공

### 데이터 처리

- 프로젝트 삭제 시 소프트 딜리트
- 계정 탈퇴 시 데이터 처리 정책 명시

### 프로젝트 상태 정의

- active: 진행 중
- completed: 완료
- locked: 플랜 다운그레이드로 잠금 (읽기 전용)

### 플랜 다운그레이드 정책 (PRO → 무료)

- 무료 한도(프로젝트 1개) 초과분은 locked 처리
- 잠긴 프로젝트는 읽기 전용 유지
- PRO 재구독 시 즉시 복구

## 2단계

- 게임화 시스템 (레벨, 퀘스트, 뱃지, 코인)
- 브라우저 익스텐션 (Chrome)

## 3단계 (검토)

- 투두 완료 시 AI 커밋 메시지 자동 생성/제안

## 수익 모델

### 무료 플랜
- 프로젝트 1개, AI 토큰 월 N, 기본 대시보드

### 유료 플랜 (PRO) - 월 3,000~5,000원
- 프로젝트 무제한, AI 토큰 월 N×5, 상세 통계, PDF 업로드

### 코인 시스템
- 퀘스트 달성 시 코인 적립 → 유료 플랜 한시적 구매 가능

## 기술 스택

| 구분 | 기술 |
|------|------|
| 프론트엔드 | Next.js 15 + TypeScript + Tailwind CSS |
| 백엔드 | Next.js API Route (풀스택) |
| DB | Supabase (PostgreSQL + Auth) |
| AI | Claude API |
| 배포 | Vercel |
| 분석 | PostHog |

## 개발 계획

- 도메인: firstep.vercel.app (초기) → firstep.io (추후)
